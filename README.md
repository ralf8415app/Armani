# Chanel & Armani – Rangliste, Bonuslevel und Rückweg

Korrekturen und neue Funktionen:
- Kein Endgegner mehr in Level 3.
- Endgegner nur noch in Level 4, etwa so groß wie die Spielfigur, 3 Treffer.
- Punkt-Rang mit 10 Speicherplätzen im Browser.
- Name kann eingetragen werden.
- Frühere Level können über „◀ Level“ erneut betreten werden, um weitere Punkte zu sammeln.
- Nach Level 3 erscheint ein Bonuslevel im Stil eines Seil-Entwirr-Puzzles.
- Im Bonuslevel können 30 Erkundungspunkte gesammelt werden.
- Punktesystem für Knochen, Herzen, Power-ups, Gegner und Boss.
