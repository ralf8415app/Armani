UPDATE: LOKALE SPEICHERUNG AUF DEM IPAD

Neu:
- Einträge werden automatisch auf dem iPad gespeichert.
- Sie bleiben nach dem Schließen und erneuten Öffnen erhalten.
- Einträge können bearbeitet und einzeln gelöscht werden.
- Alle Einträge können nur nach Sicherheitsabfrage gelöscht werden.
- XLSX-, CSV- und PDF-Export bleiben erhalten.

GitHub aktualisieren:
1. Repository Notdienst-Asyl öffnen.
2. Add file > Upload files.
3. index.html, manifest.json und sw.js hochladen.
4. Vorhandene Dateien ersetzen.
5. Commit changes drücken.
6. 2–3 Minuten warten.
7. App komplett schließen und neu öffnen.

Falls noch die alte Version erscheint:
- Home-Bildschirm-App löschen.
- Link in Safari neu öffnen.
- Wieder Zum Home-Bildschirm hinzufügen.

Wichtig:
- Safari nicht im privaten Modus verwenden.
- Beim Löschen von Safari-Websitedaten gehen lokal gespeicherte Einträge verloren.
- Regelmäßig XLSX oder CSV exportieren.
